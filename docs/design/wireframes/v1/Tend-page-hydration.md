> These wireframes show **interaction structure only**. Do not derive color, typography, spacing, or visual style from them. Follow `docs/design/brand.md` and `docs/design/visual-style.md` for all visual decisions.

---

## Flow

1. Tend home → tap **Hydration**
2. Input screen: whole number field pre-filled with the user's default amount, submit button
3. User adjusts the amount if needed → tap **Submit**
4. Success indicator → return to Tend home

Fastest path: Tend home → tap Hydration → tap Submit. Two taps to log the default amount.
